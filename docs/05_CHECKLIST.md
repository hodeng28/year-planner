# Definition of Done Checklist

## Input Format
- PR ready for review
- Feature completion

## Output Format (MUST)
- [ ] All checklist items completed
- [ ] Commands verified
- [ ] Ready for merge

---

- [ ] pnpm lint passes
- [ ] pnpm type-check passes
- [ ] pnpm build passes
- [ ] API endpoints verified with curl or http file
- [ ] Error format matches Contracts
- [ ] No duplicated types outside packages/types
- [ ] Minimal test added (e2e or unit)
- [ ] README updated if commands or env changed
